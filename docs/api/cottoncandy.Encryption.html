    AESEncryption¶

    class cottoncandy.Encryption.AESEncryption(key=None, keyfile=None, mode=2, chunk_size=65536, initialisation_vector_size=16)¶

    Bases: cottoncandy.Encryption.Encryption

    Encrypts files using an AES cipher. All files passing through the same object will be encrypted with the same key Encrypted files have the following structure:

    [ 16 bytes default ][ file size || file size % 16 - 8 bytes ][ 8 bytes ] [initialization vector][ binary ciphertext for file || padding with spaces ][file size in bytes]

    RSAAESEncryption¶

    class cottoncandy.Encryption.RSAAESEncryption(key=None, keyfile=None, mode=2, chunk_size=65536, initialisation_vector_size=16, AES_key_length=32)¶

    Bases: cottoncandy.Encryption.AESEncryption

    Encrypts each file using a unique AES key. Keys are then encrypted with RSA and returned along with the objects, the RSA private key is needed to decrypt the AES key.
